# katas

Dépôts de katas pour entrainements Python et Go.

https://www.codewars.com/kata/search/python?q=&order_by=popularity%20desc

Transformation:
```
En tant qu'expert en code et en python reprend ce kata, traduit le français, adapte le pour un exercise en python.
Fourni un script python contenant:
- en commentaire, les consignes du kata en francais
- la fonction qui permettra de résoudre le kata
- les tests unitaires avec unittest pour valider le kata
```
